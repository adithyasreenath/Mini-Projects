How to run:
	
	Run this command in commandline: 
		python map_tile.py

	Or customize the input and output paths with flags: 
		Python 
			map_tile.py
			-- latitude1 41.838928
			-- longitude1 -87.628503
			-- latitude2 41.838244
			-- longitude2 -87.626847
			-- release/debug
